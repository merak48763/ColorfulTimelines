# Colorful Timelines

## Supported Versions

- 26.1
- 26.2 (pre-2)

## Usage Example

### Step 1

Add `#colorful:sky_color` to `#in_overworld` timeline tag.

```json
{
  "values": [
    "#colorful:sky_color"
  ]
}
```

### Step 2

Use `/time` command to control the color attribute.

```mcfunction
time of colorful:sky_color set 16777215
```
